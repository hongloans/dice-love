# Dice Love (Love2D)

`Dice Love`는 `Dice War`를 Lua + Love2D로 마이그레이션한 턴 기반 점령 전략 게임입니다.  
맵은 절차적으로 생성되며, 주사위 전투를 통해 콜로니를 점령합니다.

## 1) 빠른 시작

### 요구 사항
- [Love2D 11.x](https://love2d.org/) 설치

### 실행
```bash
cd /Users/hongjiun/Documents/dev/dice-love
love .
```

## 2) 게임 목표

- 자신의 콜로니를 확장해 **모든 상대를 제거**하면 승리합니다.
- 각 콜로니는 주사위 개수를 가지며, 전투 시 각 주사위는 1~6을 굴립니다.

## 3) 기본 규칙

- 보드: `100 x 50` 그리드
- 콜로니 크기: 기본 `8`칸
- 콜로니당 최대 주사위: `20`
- 공격 가능 조건:
  - 내 콜로니 주사위가 `2` 이상
  - 대상 콜로니와 인접(상하좌우)
- 전투 결과:
  - 공격 합 > 방어 합: 점령 성공
    - 공격지 주사위 = `1`
    - 점령지 주사위 = `공격 전 주사위 - 1`
  - 그 외(동점 포함): 방어 성공
    - 공격지 주사위 = `1`
- 턴 종료 시: 내 콜로니 수만큼 주사위가 랜덤 분배됩니다(최대 20 제한).

## 4) 시작 화면 옵션

### Number of Players
- 2~8인 선택 가능

### Play Mode
- `RED PLAYER`: P1(레드) 직접 플레이 + 나머지 봇
- `ALL BOTS`: 전원 봇 자동 진행

## 5) 조작법

### 인게임
- 콜로니 클릭: 선택
- 적 콜로니 클릭: 공격 시도
- `END TURN` 버튼: 턴 종료 (플레이어 모드에서만 의미)
- `R` 버튼: 리스타트 확인 팝업
- `#` 버튼: 셀 그리드 라인 표시 토글

### 키보드
- `Space` / `Enter`: 턴 종료
- `Esc`: 리스타트 팝업 열기/닫기
- `g`: 그리드 라인 토글

## 6) 속도 단계 (Fast Forward 버튼 `>>`)

속도 버튼을 누를 때마다 순환합니다.

1. 단계 1: 느림 (관전용)
2. 단계 2: 중간
3. 단계 3: 기본 속도
4. 단계 4: 단계 3 대비 3배 빠름
5. 단계 5: **즉시 전투 반영 모드**
   - 전투 결과 연출(`battle-result`) 없이 승패가 즉시 보드에 반영됩니다.

## 7) AI 특징

현재 AI는 단순 랜덤이 아닌 다단계 판단을 사용합니다.

- 후보 공격 액션 생성
- 휴리스틱 평가 (주사위 우위, 위험/확장성, 허브 보존 등)
- 상위 후보 몬테카를로 시뮬레이션
- 기대값 기반 최종 선택
- 교착 방지:
  - 유리한 수가 없어도 합법 공격이 있으면 최선의 차선책을 강제 실행

## 8) 맵 생성 특징

- 브랜칭 기반 성장으로 가지형/뿌리형 콜로니 생성
- 후처리로 절단 콜로니 재배치
- 최종 연결성 강제 로직으로 고립 섬 컴포넌트 제거
- 콜로니 수는 과밀 방지를 위해 기존 대비 축소되어 있습니다.

## 9) 게임 종료 통계

게임 오버 화면에서 시계열 그래프를 확인할 수 있습니다.

- 점령 콜로니 수
- 총 주사위 수
- 누적 던진 주사위 수
- 누적 던진 눈 합
- 평균 눈(누적 합 / 누적 주사위)

## 11) 자주 겪는 문제

- `love: command not found`
  - Love2D가 설치되지 않았거나 PATH에 등록되지 않은 상태입니다.
- macOS 보안 경고
  - 시스템 설정에서 앱 실행 허용 후 다시 실행하세요.

## 12) 튜닝 포인트 (개발자용)

`game/constants.lua`에서 아래 항목을 주로 조정합니다.

- `MAX_COLONIES`
- `BATTLE_RESULT_DELAY`, `BOT_ACTION_DELAY`
- `SPEED_DELAY_MULT`

AI 성향은 `game/ai.lua`의 상수/가중치에서 조정 가능합니다.
